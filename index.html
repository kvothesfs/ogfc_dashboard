Understanding Shapley Values

SHAP (SHapley Additive exPlanations) values break down a prediction to show the impact of each feature. A positive value pushes the prediction higher, while a negative value pushes it lower.

The chart displays dynamically based on the current inputs in the Prediction Control Panel compared against a pre-sampled baseline of combinations. Modify parameters on the first tab and switch back to see the exact contributions recalculate.
